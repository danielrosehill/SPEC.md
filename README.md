# SPEC.md

Part of the [AI Dev Workspace Primitives](https://github.com/danielrosehill/AI-Dev-Workspace-Primitives) index.

## Purpose

A first-entry, full project specification for significant projects. Acts as a foundational stand-in for one authoritative development spec to guide the project, updated over time.

In larger projects this is eventually subsumed into a dedicated `spec/` folder with version-controlled documents; until that point, `SPEC.md` carries the authoritative spec at the repo root.
